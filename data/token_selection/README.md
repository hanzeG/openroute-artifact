This directory contains lightweight Delta-derived tables used to select the ten
paper token pairs. These are derived ranking/selection outputs, not the raw
Ripple UBRI Delta Sharing tables.

The main file used to recover the paper pair identifiers is:

```text
target_window_rankings/top15_xrp_pairs_combined_clean_20260512.csv
```

The raw AMM/CLOB/fee row exports for the full two-week experiments are much
larger and are regenerated by the README pipeline when an authorised Delta
Sharing profile is available.
